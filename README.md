## 📋 Sobre o projeto

Calculadora funcional para o navegador, com as quatro operações básicas, construída com **JavaScript Vanilla** para praticar fundamentos de manipulação do DOM e lógica de programação.

## 🛠️ Tecnologias

- **HTML5**
- **CSS3**
- **JavaScript** — sem frameworks

## 📁 Estrutura

```
.
├── index.html
└── assets/        # Imagens, ícones e estilos
```

## 🚀 Como executar

Basta abrir o arquivo `index.html` no navegador:

```bash
# Linux / macOS
xdg-open index.html

# Windows
start index.html
```

Ou rode um servidor local:

```bash
npx serve .
```

## 🎯 Aprendizados

- Manipulação do DOM com JavaScript puro
- Eventos de clique e teclado
- Estado interno sem frameworks
- Estilização de UI de calculadora
